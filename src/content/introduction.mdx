# Introduction

## What is Adex?

`adex` is a vite plugin to write full stack Node.js apps using [preact](http://preactjs.com).
It was authored to avoid writing glue code for writing server rendered preact apps. You can do most of what adex does by using existing tools,
a few examples of this can be found on [preachjs/templates](https://preachjs.github.io/templates)

## Features

- Unified Routing
- Easy to use [Islands](https://preactjs.com/blog/simplifying-islands-arch) (Opt-in feature)
- API Routes
- Works wherever Node.js does
